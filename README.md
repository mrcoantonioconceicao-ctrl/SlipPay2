# SlipPay2 🚀

Gateway de pagamento em criptomoedas para comércio eletrônico, desenvolvido em **Rust**.  
O objetivo do SlipPay2 é oferecer uma solução rápida, segura e escalável para integração de pagamentos digitais em plataformas de e-commerce.

---

## ✨ Funcionalidades
- Processamento de pagamentos em múltiplas criptomoedas.
- API RESTful para integração com sistemas externos.
- Autenticação segura via tokens.
- Logs e monitoramento de transações.
- Estrutura modular para fácil expansão.
